# 🌍 Gamified AI Habit Builder with a Living Nature World

## 🚀 Overview
A gamified AI-powered habit-building app that transforms user progress into a living nature world.

## 🎯 Features
- Habit Builder 🌱
- AI Mood Suggestions 🤖
- Dashboard with Heatmap 📊
- Achievements & Levels 🏆
- Challenge Mode ⚔️
- Streak Freeze 🧊
- Focus Timer ⏳
- Nature Growth System 🌍

## 💡 Idea
Users improve habits while growing a virtual ecosystem from cracked land to a fantasy world.

## 🧠 AI System
Provides smart suggestions based on mood and progress.

## 🔥 Unique Point
Visual nature growth + gamification + AI coaching.

## 🛠 Tech
HTML, CSS, JavaScript
